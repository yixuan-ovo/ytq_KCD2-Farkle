# 主题骰模板（配色与图标）

对局骰面 SVG 由 **`src/lib/assets/diceThemes.json`** 驱动，经 `yarn dice:generate` 输出到 `public/dice/{DieId}/`。

## 数据流

```
docs/dice/骰子模板.md（本文）
        ↓ 人工同步
src/lib/assets/diceThemes.json
        ↓ yarn dice:generate
public/dice/{DieId}/face-hidden.svg | face-1…6.svg | face-devil.svg
        ↓
src/lib/assets/diceTextures.ts → DicePiece / 对局 UI
```

**图鉴六面预览**仍使用通用骨色 [`public/dice/ivory/`](../../public/dice/ivory/)，与对局主题面分离。

## 单条主题字段

| 字段 | 说明 |
|------|------|
| `fill` | 骰面底色（HEX） |
| `border` | 描边 / 外框色 |
| `pip` | 点数圆点与图标主色 |
| `icon` | 中心装饰图标（见下表） |

每条记录的 **键名** 必须与 `diceRegistry` 中的 `DieId` 一致（如 `NormalDie`、`DevilDie`）。

## 可用 `icon` 值

与 [`src/lib/assets/diceThemes.ts`](../../src/lib/assets/diceThemes.ts) 中 `DiceIconName` 一致：

`crown` · `clover` · `devil` · `halo` · `cross` · `king` · `tooth` · `horse` · `wheel` · `skull` · `star` · `paint` · `pie` · `book` · `coin` · `strip` · `scales` · `weight` · `arrowDown` · `number3`

矢量绘制逻辑见 [`src/lib/assets/diceIcons.ts`](../../src/lib/assets/diceIcons.ts)，生成器见 [`scripts/diceSvgCore.mjs`](../../scripts/diceSvgCore.mjs)。

## 修改流程

1. 编辑 [`src/lib/assets/diceThemes.json`](../../src/lib/assets/diceThemes.json)
2. 运行 **`yarn dice:generate`**
3. 提交变更的 `diceThemes.json` 与 `public/dice/**`（若需固定产物）
4. 跑 **`yarn test`**（含 `diceThemes` / `diceTextures` 覆盖）

## 生成产物说明

| 文件名 | 用途 |
|--------|------|
| `face-hidden.svg` | 首掷前占位（「未开骰」） |
| `face-1.svg` … `face-6.svg` | 掷出后的各面（`DevilDie` 的 `face-1` 亦为恶魔图，与百搭一致） |
| `face-devil.svg` | `value === 0`（恶魔之首）；`DevilDie` 掷出 1 点亦映射为此面 |

`manifest.json` 由生成脚本写入 `public/dice/manifest.json`，供排查与测试引用。

## 骰子种类数

当前 **36** 种 `DieId`（与 `diceRegistry` / `diceThemes.json` 键数量一致）。
